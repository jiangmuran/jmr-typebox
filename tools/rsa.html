RSA

Generate an RSA key pair and encrypt/decrypt or sign/verify text — in your browser via the Web Crypto API.

How it works

Two people who can’t verify each other over a public channel can still communicate securely:

  1. Each person generates a key pair and keeps their private key secret.
  2. They exchange public keys — these are safe to share openly.
  3. To send a message, encrypt it with the recipient’s public key. Only their private key can decrypt it.
  4. Sign a message with your private key so the other side can verify it wasn’t altered.
  5. A public key alone doesn’t prove real-world identity. Compare the key fingerprint through a separate trusted channel to be sure.
